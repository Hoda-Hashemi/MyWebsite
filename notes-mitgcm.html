Build workflow

The workflow starts from a clean source tree, a chosen verification experiment, a platform-specific build option file, and an explicit generated Makefile.

git clone https://github.com/MITgcm/MITgcm.git
cd MITgcm/verification/global_oce_latlon/build
../../../tools/genmake2 -mods ../code
make depend
make
