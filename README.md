# hodahashemi.com — Discretized Flow

Portfolio and CV presenter for **Hoda Hashemi**, computational physicist (ocean dynamics, numerical modeling, HPC/CUDA).

The visual language derives from her actual research: quasi-geostrophic shallow-water dynamics, vortex-in-cell modeling, grid discretization. The hero is a **real GPU fluid solver** — a stable-fluids Navier–Stokes simulation written in raw WebGL2 (advection → curl → vorticity confinement → Jacobi pressure projection → dye transport), rendered on a visible computational lattice with ordered dithering. The cursor injects vorticity into the field.

Live site: <https://hodahashemi.com/>

## Stack

- [Astro](https://astro.build) (static output) — no framework runtime
- Raw **WebGL2** for the fluid solver (no three.js)
- **GSAP + ScrollTrigger** for scroll choreography
- **Lenis** for smooth scroll
- Live GitHub repository data pulled client-side from `api.github.com` (no auth, cached in `localStorage`)
- Fully static; no build-time secrets

## Structure

| Path | What it is |
|---|---|
| `src/pages/index.astro` | Homepage: hero, About/CV anchor, Research, Products, Writing |
| `src/pages/CV.astro` | Print-friendly light-theme CV page |
| `src/pages/notes/*.astro` | Research-note placeholders (KaTeX enabled) |
| `src/scripts/fluid-hero.js` | WebGL2 stable-fluids solver + lattice/dither display pass |
| `src/scripts/main.js` | Scroll/interaction orchestration (Lenis, GSAP, cursor, reveals) |
| `src/scripts/github-products.js` | Live GitHub repos grid (cache + fallback, MITgcm fork pinned) |
| `src/scripts/instruments.js` | 2D-canvas streamline instruments on research cards |
| `src/styles/global.css` | Design system + all section styles |
| `identity/generate_identity.py` | Dependency-free generator for the whole identity (SVG marks, icons, OG image) |
| `public/` | Generated icons, manifest, CV PDF, CNAME |

## Local development

```bash
npm install
npm run dev       # dev server
npm run build     # static build into dist/
npm run preview   # preview the build
```

## Identity

The mark is a logarithmic vortex spiral sampled onto a Cartesian grid — teal arms with a warm counter-vorticity core. All identity assets (SVG marks, favicon, app icons, OG image, webmanifest) are generated by one dependency-free script:

```bash
python3 identity/generate_identity.py
```

See [`identity/IDENTITY.md`](identity/IDENTITY.md) for the palette and asset inventory.

## Accessibility & performance

- `prefers-reduced-motion` swaps all WebGL/heavy motion for static equivalents
- Content is legible without JavaScript (reveal animations are gated behind a `.js` class)
- The fluid sim pauses when the hero is off-screen or the tab is hidden; GPU buffers are disposed on resize
- No three.js: the whole JS bundle is ~156 kB

## Deployment

GitHub Pages via `.github/workflows/deploy.yml`: every push to `main` runs `npm ci && npm run build` and deploys `dist/`. The custom domain is set through `public/CNAME`.
